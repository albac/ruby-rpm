Ruby RPM
========

Requirements
------------

+ RHEL/CentOS/Scientific
+ [EPEL](https://fedoraproject.org/wiki/EPEL)

Installation
------------

EPEL pre-setup

```
$ sudo yum install -y http://ftp.riken.jp/Linux/fedora/epel/6/i386/epel-release-6-8.noarch.rpm
```
